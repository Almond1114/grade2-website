# NEON RUSH

Three.jsで動く、ハイスピードな1人用3DアリーナFPSです。実行の簡単さよりも、操作感・敵の多さ・演出・フレームレート維持を優先しています。

## 主な内容

- WASD移動、ジャンプ、スプリント、スライド、無敵時間つき瞬間ダッシュ
- 性質の異なる3武器：VOLT RIFLE / NOVA SHOTGUN / ARC RAIL
- Runner / Drone / Bruteの3種類の敵AI
- 無限ウェーブ、コンボ、スコア、ハイスコア保存
- 爆発するエネルギーバレル、回復・エネルギードロップ
- 2ウェーブごとのランダム強化選択
- ゲージをためて発動するOVERDRIVE
- Web Audio APIによるプロシージャル効果音
- 固定タイムステップ、空間ハッシュ、オブジェクトプール、InstancedMesh
- 平均フレーム時間に応じた動的レンダリング解像度
- 外部の画像・音声・3Dモデルは不使用

## 操作

| 操作 | キー |
|---|---|
| 移動 | W / A / S / D |
| スプリント | 左Shift |
| ジャンプ | Space |
| スライド | 左Ctrl または C |
| 瞬間ダッシュ | Q |
| 射撃 | 左クリック |
| 集中照準 | 右クリック |
| リロード | R |
| 武器切替 | 1 / 2 / 3 またはマウスホイール |
| OVERDRIVE | F |
| 一時停止 | Esc |

## 起動

ES Modulesを使用しているため、ファイルを直接ダブルクリックするのではなくHTTPサーバーから開きます。

```bash
cd fps-game
python -m http.server 8000
```

ブラウザで `http://localhost:8000` を開きます。リポジトリがWebサイトとして配信されている場合は、`/fps-game/` からそのまま起動できます。

## 推奨環境

- PC向けの最新版Chrome、Edge、Firefox
- WebGL 2対応GPU
- キーボードとマウス

Three.jsはCDNから固定バージョンを読み込むため、初回起動時はインターネット接続が必要です。
